# Contributing

感谢你愿意改进这个项目。

## 本地运行

这个项目是纯前端静态网页，可以直接打开 `index.html`。如果浏览器对本地文件访问限制较多，建议在项目目录启动一个静态服务：

```bash
python3 -m http.server 8000
```

## 提交建议

- 保持纯前端可运行，不要引入必须依赖后端的功能。
- 修改通知解析时，请保留“用户确认后再写入”的流程。
- 涉及数据结构变更时，请兼容旧版 `localStorage` 数据。
- 涉及 UI 的改动，请同时检查桌面端和移动端显示。

## 数据安全

请不要在 Issue、截图或测试数据中提交真实姓名、手机号、邮箱、身份证号、申请账号、推荐人信息等敏感内容。
