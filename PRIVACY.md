# Privacy

本项目默认不连接服务器，也不会主动上传你的申请数据。

## 数据保存在哪里

数据保存在当前浏览器中：

- `localStorage`：保存主要项目数据。
- `IndexedDB`：保存一份本地镜像数据，用于提高恢复能力。
- JSON 备份：由用户手动导出并自行保存。

## 哪些信息可能比较敏感

使用时请注意，这些内容可能属于个人隐私：

- 申请学校、学院、方向和进度
- 材料清单和备注
- 官网通知链接、报名链接
- 本地材料文件夹路径
- 粘贴到通知解析框中的通知正文


## 浏览器权限

项目可能请求浏览器通知权限，用于在网页打开时提醒当天关键节点。通知权限由浏览器管理，用户可以随时在浏览器设置中关闭。

## 第三方服务

当前版本不依赖第三方后端服务。若未来接入云同步、AI 解析或账号登录，会在 README 和本文件中明确说明数据流向。
